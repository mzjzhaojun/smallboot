### 一、前言

> 项目地址：[https://gitee.com/zhengqingya/smallboot](https://gitee.com/zhengqingya/smallboot)

> 微服务版本：https://gitee.com/zhengqingya/small-tools

`SmallBoot`：Vue+SpringBoot 前后端分离的项目快速开发脚手架 & 多微信公众号管理平台 & 商城

后端技术栈：

1. SpringBoot2.7.0
2. MySQL5.7
3. MyBatis-Plus3.5.2
4. Redis6.0.8
5. Docker

前端技术栈：

1. Vue3
2. Vite4
3. Pinia
4. Vue Router
5. Element Plus
